\# block-csv2json



This project reads blockchain block data from a CSV file,

validates it, converts it into JSON, and displays it using

a simple HTML viewer.



---



\## What this project does



\- Reads block data from a CSV file

\- Validates numeric fields

\- Converts data into JSON format

\- Shows the data in a browser table
